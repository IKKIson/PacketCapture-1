

***make 명령어***
-make : 컴파일 (파일들을 합쳐준다.)
-make clean : .o파일 삭제한다.(실행파일은 삭제안함)

***참고사향***
-일단 헤더파일이랑 구조체는 stdhdr.h에 모으기
-최대한 모듈화되게끔 만드셈. 서로 주고받는 구조
-코딩할때 하기쉽게 vim 화면에 function.c와 stdhdr.h와 main.c 3개를 켜놓고 작업하기 쉽게 만듬.
 main.c파일을 vim으로 실행해서 명령어모드에서 :vs stdhdr.h        :vs function.c 하면  3개의 화면으로 분할될거임 


***코드 규칙***
-변수는 소문자로
-함수는 첫글자 대문자로
-변수나 함수 이름 선언 시 문자붙여사용하기 '_' 사용 금지.
-#define은 대문자와 '_' 사용해서 만듬 
-함수 만들 시 function.c에 구현. stdhdr.h에 함수선언부분 추가해서 기능설명 주석 달기
